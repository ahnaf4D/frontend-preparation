CSS Interview Questions

1. What is CSS and why is it important in web development?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. It is important because it allows developers to separate content from design, enabling better control over layout, colors, fonts, and overall visual aesthetics.

2. What is the box model in CSS?

The CSS box model describes the rectangular boxes generated for elements in the document tree and consists of margins, borders, padding, and the actual content area. It determines how elements are sized and spaced on a webpage.

3. What is the difference between classes and IDs in CSS?

Classes are reusable and can be applied to multiple elements, allowing for consistent styling. IDs are unique and should be applied to a single element, often used for identifying elements for JavaScript manipulation.

4. What are CSS selectors? Provide examples.

CSS selectors are patterns used to select and style elements. Examples include type selectors (e.g., p), class selectors (e.g., .class-name), ID selectors (e.g., #id-name), and attribute selectors (e.g., [type="text"]).

5. What is the difference between inline, inline-block, and block elements?

Inline elements do not start on a new line and take up only as much width as necessary (e.g., <span>). Inline-block elements are similar but allow setting width and height. Block elements start on a new line and take up the full width available (e.g., <div>).

6. How can you center a block element horizontally in CSS?

You can center a block element horizontally by setting its margin to auto: margin: 0 auto;, and specifying a width for the element.

7. What is the purpose of the z-index property in CSS?

The z-index property controls the stacking order of elements that overlap. Higher values are placed on top of lower values.

8. What are pseudo-classes and pseudo-elements in CSS? Give examples.

Pseudo-classes target elements based on their state (e.g., :hover, :focus). Pseudo-elements style specific parts of an element (e.g., ::before, ::after).

9. How do you create a flex container in CSS?

To create a flex container, set the display property of an element to flex: display: flex;. This enables flexbox layout for its children.

10. What is a CSS preprocessor and why would you use one?

A CSS preprocessor extends CSS with variables, nested rules, and functions, making it more maintainable and reusable. Examples include Sass, LESS, and Stylus.

11. What are media queries in CSS?

Media queries allow you to apply CSS rules based on the characteristics of the device or viewport, such as width, height, and orientation. They are used to create responsive designs.

12. How do you apply CSS styles to an element with a specific attribute?

You can use attribute selectors to apply styles to elements with specific attributes. For example, input[type="text"] selects all <input> elements with a type attribute value of "text".

13. What is the difference between padding and margin?

Padding is the space between the content and the border of an element, while margin is the space outside the border, between the element and its surrounding elements.

14. How do you create a CSS grid container?

To create a CSS grid container, set the display property of an element to grid: display: grid;. This enables grid layout for its children.

15. What is the purpose of the float property in CSS?

The float property is used to position elements to the left or right of their container, allowing text and inline elements to wrap around them.

16. How do you reset default browser styles in CSS?

You can reset default browser styles by using a CSS reset stylesheet (e.g., Normalize.css) or applying global styles to set default values for margins, paddings, and other properties.

17. What is the purpose of the display property in CSS?

The display property determines how an element is displayed on the page. Common values include block, inline, inline-block, flex, and grid.

18. How do you make a website responsive?

To make a website responsive, use flexible grid layouts, CSS media queries, responsive images, and frameworks like Tailwind CSS to ensure the design adapts to different screen sizes and devices.

19. What is the purpose of the position property in CSS?

The position property specifies the positioning method for an element. Values include static, relative, absolute, fixed, and sticky.

20. How do you use CSS transitions?

CSS transitions allow you to change property values smoothly over a specified duration. Example: transition: all 0.3s ease;.
