HTML Interview Questions

1. What is HTML and why is it important in web development?

HTML (HyperText Markup Language) is the standard language for creating web pages. It provides the structure of a webpage, allowing content to be organized and displayed in a web browser.

2. What are the main differences between HTML4 and HTML5?

HTML5 introduces new semantic elements (e.g., <article>, <section>), multimedia elements (e.g., <audio>, <video>), new form controls, and APIs. It also deprecates certain elements from HTML4 and provides better support for mobile devices.

3. Can you explain the structure of an HTML document?

An HTML document consists of the <!DOCTYPE html> declaration, <html>, <head>, and <body> tags. The <head> contains meta-information, and the <body> contains the visible content.

4. What are the semantic elements in HTML5? Name a few and explain their use.

Semantic elements clearly describe their meaning in a human- and machine-readable way. Examples include <header> (header section), <nav> (navigation links), <section> (generic sections), <article> (independent content), and <footer> (footer section).

5. How do you include a CSS file in an HTML document?

You include a CSS file using the <link> tag inside the <head> section: <link rel="stylesheet" href="styles.css">.

6. What is the purpose of the alt attribute in an <img> tag?

The alt attribute provides alternative text for an image if the image cannot be displayed. It also improves accessibility for screen readers.

7. How do you create a hyperlink in HTML?

You create a hyperlink using the <a> tag: <a href="https://example.com">Link text</a>.

8. What is the difference between <div> and <span>?

<div> is a block-level element used to group other elements, while <span> is an inline element used to apply styles or classes to a small part of the text.

9. Explain the difference between block-level and inline elements.

Block-level elements start on a new line and take up the full width available (e.g., <div>, <h1>). Inline elements do not start on a new line and only take up as much width as necessary (e.g., <span>, <a>).

10. How do you create a table in HTML? Explain the elements involved.

A table is created using the <table> tag. Inside it, you use <tr> for rows, <th> for header cells, and <td> for data cells.

11. What are HTML forms and which elements are commonly used in forms?

HTML forms collect user input and can include elements like <input>, <textarea>, <button>, <select>, and <label>.

12. How do you embed a video in an HTML page?

You can embed a video using the <video> tag: <video controls><source src="video.mp4" type="video/mp4">Your browser does not support the video tag.</video>.

13. What is the use of the meta tag in HTML?

The meta tag provides metadata about the HTML document, such as character set, page description, keywords, and viewport settings.

14. How do you specify character encoding in an HTML document?

You specify character encoding using a meta tag: <meta charset="UTF-8">.

15. What is the purpose of the doctype declaration in HTML?

The doctype declaration specifies the HTML version being used and helps the browser render the page correctly: <!DOCTYPE html>.

16. How do you create an unordered list and an ordered list in HTML?

An unordered list is created using <ul> and <li> tags, and an ordered list is created using <ol> and <li> tags.

17. What are data attributes and how do you use them in HTML5?

Data attributes are custom attributes prefixed with data- used to store extra information on elements: <div data-info="value"></div>.

18. How do you create a responsive layout using HTML?

Responsive layouts can be created using CSS media queries, viewport settings in the meta tag, and frameworks like Tailwind CSS.

19. What is the difference between the id and class attributes in HTML?

The id attribute uniquely identifies an element, while the class attribute can be used to apply styles to multiple elements.

20. How do you include JavaScript in an HTML document?

JavaScript can be included using the <script> tag: <script src="script.js"></script>.
